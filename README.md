Bigotry - Mine news data 
================================================


Setup 
------

1)  Create a new file in config/ called as daylife.yaml
2)  The format of the file looks like this 

    development:
      api_key:
      shared_secret:
3)  You can obtain these keys by getting a developer account daylife.com
